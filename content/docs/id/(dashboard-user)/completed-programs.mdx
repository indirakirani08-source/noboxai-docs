Transparansi adalah prioritas utama kami. Halaman ini menampilkan bukti nyata dari kebaikan yang telah Anda berikan, di mana dana yang terkumpul telah sepenuhnya disalurkan kepada penerima manfaat.

## **Daftar Program Selesai**

Ketika sebuah program telah mencapai target donasinya (100%), program tersebut akan dipindahkan ke kategori **Program Selesai Disalurkan**. Di sini, Anda dapat melihat ringkasan hasil akhir dari setiap inisiatif.

<ImageZoom
  src="/images/id/Completed%20Program.png"
  alt="Daftar Program Selesai NurimanPay"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4 my-8 mx-auto "
/>

Pada bagian ini, setiap kartu program menampilkan:
- **Label "Selesai":** Menandakan bahwa masa penggalangan dana telah berakhir.
- **Total Tersalurkan:** Jumlah dana final yang diberikan kepada penerima.
- **Indikator 100%:** Menunjukkan bahwa amanah telah ditunaikan sepenuhnya sesuai target.
- **Tombol Detail Penyaluran Dana:** Klik tombol ini untuk melihat laporan pertanggungjawaban lengkap.
