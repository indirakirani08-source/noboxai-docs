The NurimanPay homepage is designed to allow users to access essential features quickly through an efficient navigation system.

---

## **Navbar (Top Navigation Bar)**

The Navbar is located at the very top of the page and serves as the main menu for navigating between sections.

<ImageZoom
  src="/images/en/Navbar.jpeg"
  alt="NurimanPay Homepage"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4 my-8 mx-auto "
/>

* **NurimanPay Logo:** Click the logo in the top-left corner to return to the homepage.
* **Program Menu:** A dropdown menu that makes it easy to view various available donation categories.
* **About Us:** Directs you to information regarding the platform's vision, mission, and background.
* **Contact:** Information on how to reach the administrators if you have questions or issues.
* **Profile Icon:** Located in the top-right corner to access the account or log in admin only.

---

## **Hero Section**

The Hero Section is the primary visual area that users see first when opening the website. This section is designed to capture attention and provide a summary of the platform's purpose.

<ImageZoom
  src="/images/en/Hero%20Section.png"
  alt="NurimanPay Homepage"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4 my-8 mx-auto "
/>

* **Main Message:** A call-to-action phrase "Mari berbagi kebaikan bersama" (Let's share kindness together) to encourage community participation.
* **Visual Background:** Features an image of the Nurul Iman Mosque, representing the identity and primary goal of the donation distribution.
* **Donate Now Button:** A prominent Call to Action (CTA) button that directs you immediately to the list of featured donation programs.

---

<Tips>
  Use the **Navbar** if you want to find specific information regarding distribution reports, while use the **Hero Section** button for quick access to start donating.
</Tips>
